### Hint

1. 2는 1까지 3을 통해서 2단계 만에, 3까지 1단계 만에, 4까지 3을 통해서 2단계 만에, 5까지 3과 4를 통해서 3단계 만에 알 수 있다. 따라서, 케빈 베이컨의 수는 2+1+2+3 = 8이다.

- 2 -> 3 -> 4 -> 5까지 가능 최단경로는 3. 즉, 최단경로를 구하면 됨

&nbsp;

2. 5명의 유저 중에서 케빈 베이컨의 수가 가장 작은 사람은 3과 4이다.

- 최단경로 배열을 만든 다음, i=1 부터 n까지의 행 중에서, 각 행 요소들의 합이 가장 작은 유저를 탐색해내면 됨.

---

```java
import java.util.*;
import java.io.*;

class Main {

    static BufferedReader br = new BufferedReader(new InputStreamReader(System.in));
    static StringTokenizer st;
    static StringBuilder sb = new StringBuilder();
    static int n, m;
    static int INF = 100000001;
    static int[][] graph;
    public static void main(String[] args) throws IOException {
        
        st = new StringTokenizer(br.readLine());
        n = Integer.parseInt(st.nextToken());
        m = Integer.parseInt(st.nextToken());

        graph = new int[n+1][n+1];

        for(int i=1; i<=n; i++) {
            for(int j=1; j<=n; j++) {
                if(i == j) graph[i][j] = 0;
                else graph[i][j] = INF;
            }
        }

        while(m-- > 0) {
            st = new StringTokenizer(br.readLine());
            int a = Integer.parseInt(st.nextToken());
            int b = Integer.parseInt(st.nextToken());

            // 친구 관계는 양방향
            graph[a][b] = 1;
            graph[b][a] = 1;
        }

        // 플로이드 워셜
        for(int k=1; k<=n; k++) {
            for(int i=1; i<=n; i++) {
                for(int j=1; j<=n; j++) {
                    if(graph[i][j] > graph[i][k] + graph[k][j]) {
                        graph[i][j] = graph[i][k] + graph[k][j];
                    }
                }
            }
        }

        int min = Integer.MAX_VALUE;
        int idx = -1;

        // 케빈 베이컨의 수가 가장 작은 유저 탐색
        for(int i=1; i<=n; i++) {
            int tmp = 0;
            for(int j=1; j<=n; j++) {
                tmp += graph[i][j];
            }
            
            if(min > tmp) {
                min = tmp;
                idx = i;
            }
        }

        System.out.println(idx);
    }    
}
```

BFS 풀이

```java
import java.util.*;
import java.io.*;

class Main {

    static class Node {
        int v, cost;

        Node(int v, int cost) {
            this.v = v;
            this.cost = cost;
        }
    }

    static BufferedReader br = new BufferedReader(new InputStreamReader(System.in));
    static StringTokenizer st;
    static StringBuilder sb = new StringBuilder();
    static int n, m;
    static ArrayList<Node>[] graph;
    static boolean[] visited;
    static int min = 100000001;
    static int idx = -1;
    public static void main(String[] args) throws IOException {
        
        st = new StringTokenizer(br.readLine());
        n = Integer.parseInt(st.nextToken());
        m = Integer.parseInt(st.nextToken());

        graph = new ArrayList[n+1];

        for(int i=1; i<=n; i++) graph[i] = new ArrayList<>();

        while(m-- > 0) {
            st = new StringTokenizer(br.readLine());
            int a = Integer.parseInt(st.nextToken());
            int b = Integer.parseInt(st.nextToken());

            graph[a].add(new Node(b, 0));
            graph[b].add(new Node(a, 0));
        }

        for(int i=1; i<=n; i++) bfs(i);

        System.out.println(idx);

    }

    static void bfs(int st) {
        Queue<Node> Q = new LinkedList<>();
        visited = new boolean[n+1]; // 유저 방문 여부
        Q.offer(new Node(st, 0)); // 유저 시작
        visited[st] = true;

        int res = 0; // 케빈 베이컨 수

        while(!Q.isEmpty()) {
            Node cur = Q.poll();

            for(Node next : graph[cur.v]) {
                if(!visited[next.v]) {
                    visited[next.v] = true;
                    res += cur.cost + 1; // 케빈 베이컨 누적
                    Q.offer(new Node(next.v, cur.cost + 1));
                }
            }
        }

        // 케빈 베이컨 수 비교
        if(res < min) {
            min = res;
            idx = st;
        }
    }
}
```

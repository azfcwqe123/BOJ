문제: https://www.acmicpc.net/problem/14503

---

### Hint

```
1. 현재 칸이 아직 청소되지 않은 경우, 현재 칸을 청소한다.

2. 현재 칸의 주변 4칸 중 청소되지 않은 빈 칸이 없는 경우,
- 바라보는 방향을 유지한 채로 한 칸 후진할 수 있다면 한 칸 후진하고 1번으로 돌아간다.
-- 바라보는 방향의 뒤쪽 칸이 벽이라 후진할 수 없다면 작동을 멈춘다.

3. 현재 칸의 주변 4칸 중 청소되지 않은 빈 칸이 있는 경우,
- 반시계 방향으로 90도 회전한다.
-- 바라보는 방향을 기준으로 앞쪽 칸이 청소되지 않은 빈 칸인 경우 한 칸 전진한다.
--- 1번으로 돌아간다.
```
- 현재 칸에서 사방면을 탐색한 뒤, 청소가 되지 않은 곳이 있다면 이동, 없다면 후진 알고리즘을 유도해낸다.

---

```java
import java.util.*;
import java.io.*;

class Main {

    static BufferedReader br = new BufferedReader(new InputStreamReader(System.in));
    static StringTokenizer st;
    static StringBuilder sb = new StringBuilder();

    static int n, m, r, c, dir, cnt = 0;
    static int[][] map;
    static int[] dx = {-1, 0, 1, 0}; // 북 동 남 서
    static int[] dy = {0, 1, 0, -1}; 
    
    public static void main(String[] args) throws IOException {
        
        st = new StringTokenizer(br.readLine());
        n = Integer.parseInt(st.nextToken());
        m = Integer.parseInt(st.nextToken());
        map = new int[n][m];

        st = new StringTokenizer(br.readLine());
        r = Integer.parseInt(st.nextToken());
        c = Integer.parseInt(st.nextToken());
        dir = Integer.parseInt(st.nextToken());

        for(int i=0; i<n; i++) {
            st = new StringTokenizer(br.readLine());
            for(int j=0; j<m; j++) {
                map[i][j] = Integer.parseInt(st.nextToken());
            }
        }

        recur(r, c, dir);

        System.out.println(cnt);

    }    

    static void recur(int r, int c, int dir) {

        if(map[r][c] == 0) { // 청소가 안 돼있다면
            map[r][c] = -1; // 청소 표시
            cnt++;
        }

        // 0: 북쪽, 1: 동쪽, 2: 남쪽, 3: 서쪽
        // 반시계 방향: 0 -> 3 -> 2 -> 1 -> 0
        for(int d=0; d<4; d++) {
            dir = (dir + 3) % 4;

            int nx = r + dx[dir];
            int ny = c + dy[dir];

            // 청소 되지 않은 곳 발견
            if(rangeCheck(nx, ny) && map[nx][ny] == 0) {
                recur(nx, ny, dir); // 이동
                return; // 한 번 이동했으면 이후 방향은 볼 필요 없음
            }
        }

        // 사방면을 탐색했는데 청소가 다 돼있다면 후진을 한다.
        int back = (dir + 2) % 4; // 후진 방향
        int nx = r + dx[back];
        int ny = c + dy[back];
        
        // 후진이 가능하다면(벽만 아니면 이동 가능)
        if(rangeCheck(nx, ny) && map[nx][ny] != 1) {
            recur(nx, ny, dir); // 후진
        }

        // 후진 조차 못했다면?
        // 종료.
        return; // 없어도 자동 return 실행되긴함.
    }

    static boolean rangeCheck(int nx, int ny) {
        return nx >= 0 && nx < n && ny >= 0 && ny < m;
    }
}
```
